#### ** 如何克隆仓库中的某一个分支 **
```
git clone -b <branch> <remote_repo>
```
例如： git clone -b 指定的分支名字 仓库地址

**************************
#### ** 拉取远程分支到本地 **
```
git fetch origin 远程分支名x:本地分支名x
```
