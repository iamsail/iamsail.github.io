SSH Could not resolve hostname ssh.github.com

2017-07-22 • 分类: github

问题

昨天提交github,出现了如下的错误

解决

环境:

Win10
Git bash

获取到github.com的ip为192.30.255.113

注意,这个ip不是稳定不变的,以自己电脑ping出来的为准确

进入/etc目录修改hosts文件

文件末尾添加 ip github.com
